💻 Software Project

Portfolio Website

🗓 2025👤 Design & Development HTML5CSS3JavaScriptGitHub PagesFormspree

Abstract

This portfolio is a fully responsive, single-page web application built from scratch using HTML5, CSS3, and vanilla JavaScript. Designed with a cosmic purple aesthetic, it showcases my skills, projects, and contact information in an engaging and interactive format. The site features animated particle backgrounds, a custom cursor, typewriter effects, and a Formspree-integrated contact form — all crafted without any external frameworks or libraries.

Problem Statement

As an Electrical Engineering student, I needed a professional online presence that goes beyond a static résumé — something that demonstrates not just what I know, but how I build. Existing portfolio templates felt generic and failed to reflect my personal style or technical depth. The challenge was to design and develop a unique, visually distinctive portfolio from the ground up, while ensuring it remained functional, accessible, and mobile-friendly across all devices.

Methodology

The portfolio was built as a three-file project — index.html, style.css, and script.js — using only vanilla HTML, CSS, and JavaScript. A cosmic purple theme was established through CSS custom properties, while interactive features like particle animations, a custom cursor, and a typewriter effect were implemented in plain JS. The contact form was integrated with Formspree, ensuring all inputs carry the required name attributes for successful POST submissions. Responsive design is handled via media queries at three breakpoints (1024px, 768px, 480px).

Results & Outcome

The final portfolio is live and publicly accessible via GitHub Pages at devadharshni57.github.io/Portfolio. All interactive features — particles, custom cursor, and typewriter effect — render correctly across modern browsers. The contact form successfully routes submissions to the designated email address. The layout adapts fluidly across desktop, tablet, and mobile screen sizes, delivering a consistent user experience. The project served as a hands-on exercise in debugging JavaScript scope errors, CSS specificity conflicts, and third-party form integration — strengthening both technical and problem-solving skills.

QR-Based Bike Ignition System

🗓 2026👤 Circuit integration & testing QR AuthESP32Circuit

Abstract

Developed a secure keyless bike ignition system integrating hardware and software using QR authentication and OTP verification through a mobile app or web platform. The system enhances vehicle security and supports modern connected automotive solutions.

Problem Statement

Traditional ignition systems are vulnerable to theft and duplication. This project addresses these issues by enabling secure digital authentication. It improves safety, convenience, and unauthorized access prevention.

System Architecture

Used ESP32 for secure communication and ignition control. The QR code redirects users to a mobile or web login page, followed by OTP verification. After authentication, the ignition circuit is activated.

Results & Outcome

Achieved secure and fast authentication with reliable hardware & software integration. It ensures user convenience and reliablity.
